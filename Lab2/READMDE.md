# Lab 2 – Web Development

This folder contains the files and code for **Lab 2** of the Web Development course.

## Objective
The goal of this lab is to understand the basic structure of an HTML webpage and apply basic CSS styling.

## Files Included
- **index.html** – The main HTML file that contains the webpage structure.
- **style.css** – The CSS file used to style the webpage.
- **README.md** – This file provides information about the contents and purpose of Lab 2.

## Description
In this lab, we created a simple webpage using HTML and CSS.  
The page includes:
- A heading and a short paragraph  
- Some basic text formatting  
- CSS for colors, fonts, and layout  

## Outcome
After completing this lab, I learned how to:
- Write and organize HTML code  
- Link a CSS file to an HTML page  
- Use basic CSS properties to style a webpage
